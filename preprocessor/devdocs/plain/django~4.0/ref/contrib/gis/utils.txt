# GeoDjango Utilities

The `django.contrib.gis.utils` module contains various utilities that are
useful in creating geospatial web applications.

  * `LayerMapping` data import utility
    * Example
    * `LayerMapping` API
    * Troubleshooting
  * OGR Inspection
    * `ogrinspect`
    * `mapping`
  * `GeoJSON` Serializer
