# tf.experimental.numpy.random.seed

TensorFlow variant of NumPy's `random.seed`.

    
    tf.experimental.numpy.random.seed(
        s
    )
    
Sets the seed for the random number generator.

Uses `tf.set_random_seed`.

Args: s: an integer.

See the NumPy documentation for `numpy.random.seed`.
