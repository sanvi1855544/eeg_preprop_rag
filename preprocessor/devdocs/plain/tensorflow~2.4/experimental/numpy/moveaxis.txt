# tf.experimental.numpy.moveaxis

TensorFlow variant of NumPy's `moveaxis`.

    
    tf.experimental.numpy.moveaxis(
        a, source, destination
    )
    
Raises ValueError if source, destination not in (-ndim(a), ndim(a)).

See the NumPy documentation for `numpy.moveaxis`.
