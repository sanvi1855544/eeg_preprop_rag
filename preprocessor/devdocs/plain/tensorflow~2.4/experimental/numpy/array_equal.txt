# tf.experimental.numpy.array_equal

TensorFlow variant of NumPy's `array_equal`.

    
    tf.experimental.numpy.array_equal(
        a1, a2
    )
    
Unsupported arguments: `equal_nan`.

See the NumPy documentation for `numpy.array_equal`.
