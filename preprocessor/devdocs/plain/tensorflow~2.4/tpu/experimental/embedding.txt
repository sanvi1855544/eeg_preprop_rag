# Module: tf.tpu.experimental.embedding

Public API for tf.tpu.experimental.embedding namespace.

## Classes

`class Adagrad`: Optimization parameters for Adagrad with TPU embeddings.

`class Adam`: Optimization parameters for Adam with TPU embeddings.

`class FeatureConfig`: Configuration data for one embedding feature.

`class SGD`: Optimization parameters for stochastic gradient descent for TPU
embeddings.

`class TPUEmbedding`: The TPUEmbedding mid level API.

`class TableConfig`: Configuration data for one embedding table.

## Functions

`serving_embedding_lookup(...)`: Apply standard lookup ops with
`tf.tpu.experimental.embedding` configs.
