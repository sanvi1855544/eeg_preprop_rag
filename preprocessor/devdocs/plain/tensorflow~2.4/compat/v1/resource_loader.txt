# Module: tf.compat.v1.resource_loader

Resource management library.

## Functions

`get_data_files_path(...)`: Get a direct path to the data files colocated with
the script.

`get_path_to_datafile(...)`: Get the path to the specified file in the data
dependencies.

`get_root_dir_with_all_resources(...)`: Get a root directory containing all
the data attributes in the build rule.

`load_resource(...)`: Load the resource at given path, where path is relative
to tensorflow/.

`readahead_file_path(...)`: Readahead files not implemented; simply returns
given path.
