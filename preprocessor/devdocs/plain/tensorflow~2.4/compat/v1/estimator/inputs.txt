# Module: tf.compat.v1.estimator.inputs

Utility methods to create simple input_fns.

## Functions

`numpy_input_fn(...)`: Returns input function that would feed dict of numpy
arrays into the model.

`pandas_input_fn(...)`: Returns input function that would feed Pandas
DataFrame into the model.
