# Module: tf.compat.v1.nest

Public API for tf.nest namespace.

## Functions

`assert_same_structure(...)`: Asserts that two structures are nested in the
same way.

`flatten(...)`: Returns a flat list from a given nested structure.

`is_nested(...)`: Returns true if its input is a collections.abc.Sequence
(except strings).

`map_structure(...)`: Applies `func` to each entry in `structure` and returns
a new structure.

`pack_sequence_as(...)`: Returns a given flattened sequence packed into a
given structure.
