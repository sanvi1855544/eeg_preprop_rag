# tf.inside_function

Indicates whether the caller code is executing inside a `tf.function`.

    
    tf.inside_function()
    
Returns  
---  
Boolean, True if the caller code is executing inside a `tf.function` rather
than eagerly.  
#### Example:

    
    tf.inside_function()
    False
    @tf.function
    def f():
      print(tf.inside_function())
    f()
    True
