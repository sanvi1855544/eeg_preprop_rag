# tf.math.logical_or

Returns the truth value of x OR y element-wise.

#### View aliases

Main aliases

`tf.logical_or`

Compat aliases for migration

See Migration guide for more details.

        x, y, name=None
    )

---

Returns

A `Tensor` of type `bool`.
