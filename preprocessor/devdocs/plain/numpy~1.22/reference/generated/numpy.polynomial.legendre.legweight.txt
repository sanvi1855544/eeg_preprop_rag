# numpy.polynomial.legendre.legweight

polynomial.legendre.legweight(x)[source]

    
Weight function of the Legendre polynomials.

The weight function is \\(1\\) and the interval of integration is \\([-1,
1]\\). The Legendre polynomials are orthogonal, but not normalized, with
respect to this weight function.

Parameters

    
xarray_like

    
Values at which the weight function will be computed.

Returns

    
wndarray

    
The weight function at `x`.
