# numpy.dtype.isalignedstruct

attribute

dtype.isalignedstruct

    
Boolean indicating whether the dtype is a struct which maintains field
alignment. This flag is sticky, so when combining multiple structs together,
it is preserved and produces new dtypes which are also aligned.
