# numpy.char.isupper

char.isupper(a)[source]

    
Returns true for each element if all cased characters in the string are
uppercase and there is at least one character, false otherwise.

Call `str.isupper` element-wise.

For 8-bit strings, this method is locale-dependent.

Parameters

    
aarray_like of str or unicode

Returns

    
outndarray

    
Output array of bools
