# numpy.distutils.ccompiler_opt.CCompilerOpt.dist_info

method

distutils.ccompiler_opt.CCompilerOpt.dist_info()[source]

    
Return a tuple containing info about (platform, compiler, extra_args),
required by the abstract class ‘_CCompiler’ for discovering the platform
environment. This is also used as a cache factor in order to detect any
changes happening from outside.
