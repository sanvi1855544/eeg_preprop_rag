# numpy.char.replace

char.replace(a, old, new, count=None)[source]

    
For each element in `a`, return a copy of the string with all occurrences of
substring `old` replaced by `new`.

Calls `str.replace` element-wise.

Parameters

    
aarray-like of str or unicode

old, newstr or unicode

countint, optional

    
If the optional argument `count` is given, only the first `count` occurrences
are replaced.

Returns

    
outndarray

    
Output array of str or unicode, depending on input type
