# numpy.recarray.reshape

method

recarray.reshape(shape, order='C')

    
Returns an array containing the same data with a new shape.

Refer to `numpy.reshape` for full documentation.

See also

`numpy.reshape`

    
equivalent function

#### Notes

Unlike the free function `numpy.reshape`, this method on `ndarray` allows the
elements of the shape parameter to be passed in as separate arguments. For
example, `a.reshape(10, 11)` is equivalent to `a.reshape((10, 11))`.
