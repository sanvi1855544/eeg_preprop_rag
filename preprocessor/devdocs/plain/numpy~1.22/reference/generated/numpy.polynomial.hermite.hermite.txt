# numpy.polynomial.hermite.Hermite

class numpy.polynomial.hermite.Hermite(coef, domain=None, window=None)[source]

    
An Hermite series class.

The Hermite class provides the standard Python numerical methods ‘+’, ‘-’,
‘*’, ‘//’, ‘%’, ‘divmod’, ‘**’, and ‘()’ as well as the attributes and methods
listed in the `ABCPolyBase` documentation.

Parameters

    
coefarray_like

    
Hermite coefficients in order of increasing degree, i.e, `(1, 2, 3)` gives
`1*H_0(x) + 2*H_1(X) + 3*H_2(x)`.

domain(2,) array_like, optional

    
Domain to use. The interval `[domain[0], domain[1]]` is mapped to the interval
`[window[0], window[1]]` by shifting and scaling. The default value is [-1,
1].

window(2,) array_like, optional

    
Window, see `domain` for its use. The default value is [-1, 1].

New in version 1.6.0.
