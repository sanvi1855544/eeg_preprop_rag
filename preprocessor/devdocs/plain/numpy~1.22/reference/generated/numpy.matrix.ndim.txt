# numpy.matrix.ndim

attribute

matrix.ndim

    
Number of array dimensions.

#### Examples

    
    >>> x = np.array([1, 2, 3])
    >>> x.ndim
    1
    >>> y = np.zeros((2, 3, 4))
    >>> y.ndim
    3
