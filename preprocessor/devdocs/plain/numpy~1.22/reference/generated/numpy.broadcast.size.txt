# numpy.broadcast.size

attribute

broadcast.size

    
Total size of broadcasted result.

#### Examples

    
    >>> x = np.array([1, 2, 3])
    >>> y = np.array([[4], [5], [6]])
    >>> b = np.broadcast(x, y)
    >>> b.size
    9
