# numpy.distutils.ccompiler_opt.CCompilerOpt.conf_features_partial

method

distutils.ccompiler_opt.CCompilerOpt.conf_features_partial()[source]

    
Return a dictionary of supported CPU features by the platform, and accumulate
the rest of undefined options in `conf_features`, the returned dict has same
rules and notes in class attribute `conf_features`, also its override any
options that been set in ‘conf_features’.
