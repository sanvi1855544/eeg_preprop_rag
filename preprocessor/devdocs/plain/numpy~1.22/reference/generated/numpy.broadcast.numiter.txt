# numpy.broadcast.numiter

attribute

broadcast.numiter

    
Number of iterators possessed by the broadcasted result.

#### Examples

    
    >>> x = np.array([1, 2, 3])
    >>> y = np.array([[4], [5], [6]])
    >>> b = np.broadcast(x, y)
    >>> b.numiter
    2
