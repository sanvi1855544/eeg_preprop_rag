# matplotlib.axes.Axes.set_xbound

Axes.set_xbound(lower=None, upper=None)[source]

    
Set the lower and upper numerical bounds of the x-axis.

This method will honor axis inversion regardless of parameter order. It will
not change the autoscaling setting (`get_autoscalex_on()`).

Parameters

    
lower, upperfloat or None

    
The lower and upper bounds. If None, the respective axis bound is not
modified.
