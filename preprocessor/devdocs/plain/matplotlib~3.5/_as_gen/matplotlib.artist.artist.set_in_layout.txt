# matplotlib.artist.Artist.set_in_layout

Artist.set_in_layout(in_layout)[source]

    
Set if artist is to be included in layout calculations, E.g. Constrained
Layout Guide, `Figure.tight_layout()`, and `fig.savefig(fname,
bbox_inches='tight')`.

Parameters

    
in_layoutbool
