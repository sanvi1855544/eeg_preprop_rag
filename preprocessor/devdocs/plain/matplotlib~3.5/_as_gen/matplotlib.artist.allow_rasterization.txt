# matplotlib.artist.allow_rasterization

matplotlib.artist.allow_rasterization(draw)[source]

    
Decorator for Artist.draw method. Provides routines that run before and after
the draw call. The before and after functions are useful for changing artist-
dependent renderer attributes or making other setup function calls, such as
starting and flushing a mixed-mode renderer.
