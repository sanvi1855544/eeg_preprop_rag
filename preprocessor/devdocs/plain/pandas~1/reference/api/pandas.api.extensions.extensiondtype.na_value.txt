# pandas.api.extensions.ExtensionDtype.na_value

propertyExtensionDtype.na_value

    
Default NA value to use for this type.

This is used in e.g. ExtensionArray.take. This should be the user-facing
“boxed” version of the NA value, not the physical NA value for storage. e.g.
for JSONArray, this is an empty dictionary.
