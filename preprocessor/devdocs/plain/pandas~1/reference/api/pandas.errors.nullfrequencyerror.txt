# pandas.errors.NullFrequencyError

exceptionpandas.errors.NullFrequencyError[source]

    
Error raised when a null freq attribute is used in an operation that needs a
non-null frequency, particularly DatetimeIndex.shift, TimedeltaIndex.shift,
PeriodIndex.shift.
