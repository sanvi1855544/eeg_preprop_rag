# pandas.Timestamp.normalize

Timestamp.normalize()

    
Normalize Timestamp to midnight, preserving tz information.

Examples

    
    >>> ts = pd.Timestamp(2020, 3, 14, 15, 30)
    >>> ts.normalize()
    Timestamp('2020-03-14 00:00:00')
