# pandas.Timestamp.is_quarter_start

Timestamp.is_quarter_start

    
Return True if date is first day of the quarter.

Examples

    
    >>> ts = pd.Timestamp(2020, 3, 14)
    >>> ts.is_quarter_start
    False
    
    
    >>> ts = pd.Timestamp(2020, 4, 1)
    >>> ts.is_quarter_start
    True
