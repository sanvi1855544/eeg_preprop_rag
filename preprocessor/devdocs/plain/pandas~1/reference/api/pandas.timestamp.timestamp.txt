# pandas.Timestamp.timestamp

Timestamp.timestamp()

    
Return POSIX timestamp as float.

Examples

    
    >>> ts = pd.Timestamp('2020-03-14T15:32:52.192548')
    >>> ts.timestamp()
    1584199972.192548
