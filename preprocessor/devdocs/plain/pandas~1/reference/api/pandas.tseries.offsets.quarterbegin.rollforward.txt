# pandas.tseries.offsets.QuarterBegin.rollforward

QuarterBegin.rollforward()

    
Roll provided date forward to next offset only if not on offset.

Returns

    
TimeStamp

    
Rolled timestamp if not on offset, otherwise unchanged timestamp.
