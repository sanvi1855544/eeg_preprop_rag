# torch.utils.get_cpp_backtrace

`torch.utils.get_cpp_backtrace(frames_to_skip=0, maximum_number_of_frames=64)`
[source]

    
Returns a string containing the C++ stack trace of the current thread. :param
frames_to_skip: the number of frames to skip from the top of the stack :type
frames_to_skip: int :param maximum_number_of_frames: the maximum number of
frames to return :type maximum_number_of_frames: int

Return type

    
str
