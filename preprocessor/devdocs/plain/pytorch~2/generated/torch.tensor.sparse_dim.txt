# torch.Tensor.sparse_dim

`Tensor.sparse_dim() → int`

    
Return the number of sparse dimensions in a sparse tensor `self`.

Note

Returns `0` if `self` is not a sparse tensor.

See also `Tensor.dense_dim()` and hybrid tensors.
